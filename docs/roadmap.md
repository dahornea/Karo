# Karo Roadmap

## Phase 1 - Core Multiplayer Foundation

Status: Completed

- Private lobby rooms, player presence, host start, and synchronized shared state

## Phase 2 - Base Game Rules

Status: Completed

- Setup, turns, dice production, construction, maritime and player trade, Warden flow, and Development Cards

## Phase 3 - Rule Completion

Status: Completed, with match-end presentation polish remaining

- Largest Army, Longest Trail, 10 Victory Point validation, and authoritative match completion

## Phase 4 - UX and Portfolio Polish

Status: Completed for the current MVP

- Board-first responsive UI, progressive-disclosure drawers, polished two-column Development Cards hand, premium symbolic card family, icon and asset system, and a curated six-image portfolio gallery

## Phase 5 - Persistence and Production Architecture

Status: Planned

- Database-backed sessions and matches, durable reconnect, match history, player statistics, CI, broader automated integration coverage, and deployment documentation

## Phase 6 - Future Play Modes

Status: Planned

- Bots, spectators, replays, custom cards, and custom maps or modes
